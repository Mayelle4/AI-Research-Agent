# AI Research Agent

An AI research assistant using LangChain and Anthropic for generating structured research summaries.  

## Features
- Uses Anthropic Claude API for LLM responses.
- Supports Wikipedia and DuckDuckGo search tools.
- Outputs structured research responses in JSON format.

## Requirements
- Python 3.11+
- Packages listed in `requirements.txt`

## Usage
```bash
pip install -r requirements.txt
python main.py
